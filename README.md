# Snack Ordering Platform

Production-ready MERN stack online snack ordering website.

## Apps

- `client`: React 19, Vite, TypeScript, Redux Toolkit, RTK Query, Tailwind CSS, shadcn/ui.
- `server`: Node.js, Express, MongoDB, Mongoose, JWT auth, Razorpay, Cloudinary.

## Phase 1

This phase establishes the scalable project structure and foundational app boundaries.
